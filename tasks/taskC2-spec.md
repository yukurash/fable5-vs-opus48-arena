# RetroDesk — ブラウザで動く Windows 95 風デスクトップ 要件

## 技術要件

- 静的な HTML/CSS/JS のみ。`index.html` を開けば動く(ビルド工程なし)
- 外部ライブラリの CDN 読み込みは可

## 必須機能

### デスクトップ

- [ ] Windows 95 風の見た目(配色・ベベル・フォント感)
- [ ] デスクトップアイコンが並んでいる(ダブルクリックでアプリが開く)
- [ ] アイコンはドラッグで移動できる

### ウィンドウシステム

- [ ] ウィンドウはタイトルバーのドラッグで移動できる
- [ ] 最小化 / 最大化 / 閉じるボタンが機能する
- [ ] 複数ウィンドウの重なり順制御(クリックしたウィンドウが前面に来る)

### タスクバー

- [ ] スタートボタンとスタートメニュー(メニューからアプリを起動できる)
- [ ] 開いているウィンドウがタスクバーに表示され、クリックで最小化/復元
- [ ] 右端に現在時刻の時計

### アプリ(最低3つ、すべて動作すること)

- [ ] **ペイント**: マウスで描ける、色を選べる、クリアできる
- [ ] **マインスイーパー**: 実際に遊べる(地雷配置、数字表示、旗、勝敗判定)
- [ ] **メモ帳**: テキスト編集ができ、localStorage に保存・復元できる

### ゴミ箱

- [ ] デスクトップアイコンをゴミ箱にドラッグすると削除される
- [ ] ゴミ箱を開くと削除したアイコンが見え、元に戻せる

## 自由演出

起動画面、起動音、エラーダイアログ、スクリーンセーバー、イースターエッグなど、世界観を高める演出は自由に追加してよい。
